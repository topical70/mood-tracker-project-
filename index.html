Class 12 AI Project

Title: Mood Tracking App with Data Visualization

Problem Features AI & Data SDGs Future Scope

1. Problem Statement

This project focuses on building a simple mood tracking app for students. The app will help users log their mood daily and visualize the data over time (e.g., Happy, Sad, Angry).

2. Users/Stakeholders

  • Students
  • Educators

3. Objectives

  • Track mood consistently for a week or month.
  • Visualize mood data through charts.
  • Understand how data analysis helps in self-awareness.

4. Key Features

  • Mood logging system (daily entry).
  • Simple dashboard showing mood patterns using graphs.
  • Basic HTML interface with a guide for students to build it.

5. AI Used

No complex AI is used in this version. The focus is on understanding data visualization and user interaction.

6. Data Used

  • Self-generated mood data (Happy, Sad, Angry).
  • Collected over a set period (week/month).

7. Sustainable Development Goals (SDGs)

  • Goal 3 – Good Health and Well-being: Encourages mental health tracking.
  • Goal 4 – Quality Education: Enhances data literacy and tech usage in students.

8. Future Scope

  • Use AI/ML to predict mood trends.
  • Enable mood-based suggestions for mental wellness.
  • Expand with user feedback and more mood options.
